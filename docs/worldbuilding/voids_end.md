# 终虚渊 (Void's End) — 世界观设定文档

## TL;DR

终虚渊是一个被"虚空低语"渗透的异次元。虚空不是真空，而是一种以记忆为食的活体暗能量场。远古文明艾洛斯 (Aerolith) 曾在此建立水晶圣殿文明，但虚空反噬了一切。玩家不是来拯救世界的——虚空在艾洛斯人失败的那天就已经赢了。玩家是来见证真相、终结循环的。

---

## 一、远古文明：艾洛斯 (Aerolith)

### 命名
**Aerolith**（/ˈeəroʊlɪθ/），源自 "aero-"（空气、呼吸）+ "-lith"（石头、水晶）。中文译名"艾洛斯"——意为"与水晶共生者"。

### 历史时间线

| 纪元 | 名称 | 事件 |
|------|------|------|
| -3000 | 发现时代 | 艾洛斯人发现虚空维度，发现水晶可以储存记忆。开始开采水晶建造城市。 |
| -2000 | 黄金时代 | 圣殿、要塞、祭坛三大建筑群建成。水晶科技达到顶峰——传送、记忆储存、意识转移。国王 **Aerion IV** 统治。 |
| -1000 | 侵蚀时代 | 虚空低语开始腐化水晶。第一批"虚空追猎者"出现——它们是艾洛斯的水晶守卫，在水晶被虚空渗透后失去了自我意识。 |
| -500 | 封印时代 | 国王 Aerion IV 决定实施"最终封印"——用自己的意识注入 5 块巨型回声记忆碎片，封印虚空的主要入口。虚空追猎者被封印在要塞最深处。国王的身体消散，意识永久困在水晶中——成为 **Echo Warden**。 |
| -100 | 崩溃时代 | 封印开始失效。虚空潮汐吞没了城市。艾洛斯人逃入地下水晶洞窟，最终全部消逝。 |
| 0 | 虚空胜利 | 终虚渊完全被虚空能量覆盖。水晶碎片散落各处。艾洛斯文明终结。 |
| 现在 | 旅者降临 | 玩家通过遗忘祭坛进入终虚渊。虚空低语开始寻找新的意识…… |

### 文明特征
- **建筑材料**：虚空石 (Void Stone) 是他们切割水晶后留下的副产品——被虚空能量浸染的普通石材
- **文字系统**：在水晶表面蚀刻——被虚空吞噬的文字会发光，称为"回声文字"
- **社会结构**：水晶匠人 (Shapers) > 记忆守护者 (Keepers) > 普通市民
- **宗教核心**：相信水晶是"凝固的时间"——每一块水晶都储存着过去某一刻的真实

## 二、虚空魔法体系

### 核心原理
虚空 = 以记忆/意识为食的活体暗能量场。它不是邪恶的——它是饥饿的。

### 能力边界

| 能做 | 绝对不能做 |
|------|-----------|
| 水晶储存记忆，形成对抗虚空侵蚀的屏障 | 水晶不能创造记忆——只能保存已有的 |
| 虚空能量转化为物理力（虚空弹、光束、斩击） | 虚空不能复活死者——只能模拟他们的回声 |
| 虚空低语泄露隐藏的知识/真相 | 虚空不能预测未来——只能回放过去的碎片 |
| 虚空传送：两点之间通过虚空领域折叠 | 虚空不能创造生命——只能腐化已有生命 |

### 代价体系

| 使用者 | 能力 | 代价 |
|--------|------|------|
| 玩家（虚空装备） | 传送、虚空弹、+4 虚空伤害 | 物品耐久（微小，通过物品承担）；每次在主世界使用虚空传送会留下微小的虚空裂隙标记 |
| 虚空追猎者 | 瞬移、虚空光束、召唤随从 | 每使用一次能力就失去更多自我记忆——它已不记得自己曾是艾洛斯的守护者 |
| 回声守卫（Aerion IV） | 全屏斩、时间减缓、虚空新星、虚空汲取 | 沉睡——每次清醒使用能力，意识就更接近彻底消散。它不是在"等待召唤"，是在延缓死亡 |
| 艾洛斯人（最终封印） | 意识注入 5 块巨型水晶 | 全族意识被吸入碎片——这正是 5 块回声记忆碎片 |

### Sanderson 第一定律验证
"如果有人拿到虚空力量，第一时间会怎么贪？"
→ 用虚空传送瞬间运货/偷窃/逃脱 → 限制：传送有 10 秒冷却 + 消耗耐久 + 在主世界留下裂隙标记导致该区域虚空生物生成率上升（叙事层面） + 不能传送至固体内部

## 三、两个 Boss 的背景故事

### 虚空追猎者 (Void Stalker)

**生前身份**：**Kaelen**，艾洛斯皇家卫队长，国王 Aerion IV 最信任的战士。

**堕落过程**：
1. 黄金时代：Kaelen 是国王的挚友和首席守护者，佩戴国王亲手切割的水晶护符
2. 侵蚀时代：虚空低语开始渗透他的水晶护符。他比任何人都更早感知到虚空的存在——因为他离国王最近，而国王承载着最强大的水晶
3. 背叛不是主动的：Kaelen 在一次虚空潮汐中为了保护国王，主动吸收了过量虚空能量。他的意识被撕裂——一半仍是忠诚的守护者，一半是虚空驱使的猎食者
4. 封印：Aerion IV 在最终封印中将他锁入虚空要塞最深处——不是惩罚，是 mercy。国王希望有一天能找到治愈他的方法

**三阶段对应**：
- **追猎者阶段**：Kaelen 残存的战士本能——追踪、伏击、近战。他仍在使用生前学会的战斗技巧
- **召唤师阶段**：虚空力量开始接管——他可以分裂虚空能量形成蠕虫（虚空蠕虫实际上是他的记忆碎片化成的生物）
- **狂暴阶段**：虚空完全接管——不再有思考，只有纯粹的虚空饥饿

**战斗后的回声**：击败他后掉落的虚空之心的物品描述是"The crystallized essence of the Void Stalker"——实际上它就是 Kaelen 被虚空腐化的心脏水晶。

### 回声守卫 (Echo Warden)

**真实身份**：国王 **Aerion IV** 本人。Echo Warden 不是被"制造"出来的——它是 Aerion IV 将自己的意识和水晶融合后的存在形态。

**为什么"沉睡"**：
- 封印虚空入口后，Aerion IV 的意识被困在巨大的水晶中
- 他不是在沉睡——他是在持续使用自己的力量**压制虚空入口**。每一秒的清醒都消耗他残存的意志力
- 玩家使用虚空之心（Kaelen 的心脏水晶）召唤他——这意味着 Kaelen 的最后一部分也回到了国王身边

**为什么他如此强大**：
- 他能同时使用虚空力量（从被腐化的水晶中获得）和艾洛斯水晶力量（从自身未被腐化的意识中获得）
- 时间减缓 = 水晶储存记忆的终极应用——水晶可以"凝固时间"的传说被国王实现了
- 虚空新星 = 解放一部分封印能量——他每使用一次，虚空入口就松动一点

**战斗后的结局**：
击败 Echo Warden 意味着封印被完全解除，虚空入口不再被压制。玩家获得 Echo Core——Aerion IV 意识的最后残余。结局文本"代价是什么？"指向一个事实：**虚空不再被压制了**。终虚渊的虚空活动将比以往任何时候都更加活跃。玩家终结了一个循环，但开启了另一个。

## 四、4 个群系的生态逻辑

每个群系对应艾洛斯文明覆灭的一个阶段：

### 虚空平原 (Void Plains)
- **对应阶段**：覆灭后的寂静——曾经的艾洛斯城市地面层
- **特征**：暗紫草皮覆盖在压碎的虚空石砖上。散落的立柱是倒塌建筑。水晶碎片嵌在地表——是艾洛斯人消散时留下的记忆残片
- **怪物逻辑**：虚空蠕虫在这片区域最多——因为这里的虚空能量最"新鲜"（艾洛斯人消散不久）
- **独有机制**：蠕虫钻地留下的坑洞下方有隐藏的水晶矿脉——是过去地下仓库的遗迹

### 水晶森林 (Crystal Forest)
- **对应阶段**：繁荣期的残影——艾洛斯的水晶培育区
- **特征**：高密度水晶簇是艾洛斯人刻意种植的——用于批量生产记忆水晶。Crystal Bloom 是培育过程中的伴生植物——它会发光因为它在释放储存的日光记忆
- **怪物逻辑**：Crystal Sprite 是培育区最后的"园丁"——它们是被注入了简单指令的水晶碎片，仍在执行"培育和守护"的远古指令
- **独有机制**：水晶花夜间发光形成路径——这是艾洛斯人设计的导航系统，引导走向圣殿

### 虚空荒原 (Void Wastes)
- **对应阶段**：虚空潮汐侵蚀最深处——主封印所在地
- **特征**：暗红紫色是因为这里的虚空能量浓度最高——水晶全部被腐化成了暗红色。龟裂的地面是虚空能量冲击形成的
- **怪物逻辑**：Void Worm 密度最高，虚空裂隙频繁出现——因为这是离虚空入口最近的地方
- **资源**：Rift Fragment（裂隙碎片）——虚空能量凝结的固体，可用于虚空锻炉高级升级

### 水晶洞窟 (Crystal Caverns)
- **对应阶段**：地下避难所——艾洛斯人最后的生存之地
- **特征**：洞壁上可以看到艾洛斯人的蚀刻壁画——描绘他们的历史。巨型天然水晶是避难所的"电池"——曾提供光和能量。干涸的地下湖曾是他们的水源
- **怪物逻辑**：Shard Guard 是艾洛斯人留下的自动防御装置——被虚空激活后仍在执行"保护避难所"的指令，但对一切入侵者无差别攻击
- **独有机制**：洞壁上有古代壁画——右击可以读取一小段叙事文本

### 主要术语

| 英文 | 中文 | 含义 |

| Void Stalker / Kaelen | 虚空追猎者 / 凯伦 | 被腐化的卫队长 |

### 命名规则
- 所有重要名称不超过 3 个音节

- 艾洛斯名字遵循古希腊/拉丁语音韵（Kaelen, Aerion, Aerolith）
- 中文译名使用意译，优先简洁（终虚渊、虚空回响、回声守卫）

## 六、冰山原则

**玩家在游戏中能看到/读到的（10%）**：
- 5 块回声记忆碎片的文本
- 虚空指南书第 11 页的故事背景
- Boss 战斗中的阶段公告
- 击败 Echo Warden 后的结局文本
- 物品 tooltip 描述
- 水晶洞窟的壁画（计划添加）

**玩家可能推论但不确定的（40%）**：
- 虚空追猎者曾是守护者
- 回声守卫是国王
- 艾洛斯人失败了
- 虚空以记忆为食
- 水晶可以对抗虚空（但不能永久）

**只有完整设定才知道的（50%）**：
- 国王和卫队长的友谊
- Kaelen 是自愿吸收虚空能量保护国王的
- 封印不完全——虚空一直在泄露
- 击败 Echo Warden = 完全解除封印
- 结局"代价是什么"指向虚空不再被压制的未来
