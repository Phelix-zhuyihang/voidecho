## 一、模组概述 / Overview

**Void Echo: Ancient Sanctum（虚空回响：远古圣殿）** 是一个以 **虚空维度探索 + 多阶段 Boss 讨伐** 为核心玩法的 Fabric 冒险模组。玩家将从主世界发现远古遗迹开始，穿越到名为 **"终虚渊（Void's End）"** 的异世界维度，在四大群系中冒险，击败两大 Boss，打造强大的虚空合金装备，最终揭开古代文明的秘密。

### 核心特色 / Key Features

| 系统 | 内容 |
|------|------|

### 3.2 四大群系 / Four Biomes

#### 🌌 虚空平原（Void Plains）
- **温度/湿度:** 0.5 / -0.5
- **色调:** 蓝色调
- **地形:** 平坦的浮空岛屿，虚空草方块地表
- **怪物:** 虚空蠕虫（weight 100）、暗晶幽灵（weight 50）、**虚空追猎者**（weight 5）
- **特征:** 水晶矿石（地下）、虚空草皮覆盖
- **结构:** 虚空堡垒（地下城）
- **策略:** 最常见群系，适合初期探索和刷水晶碎片

#### 💠 暗晶森林（Crystal Forest）
- **温度/湿度:** -0.5 / 0.5
- **色调:** 蓝绿色调
- **地形:** 相对低洼的谷地
- **怪物:** 虚空蠕虫 + 暗晶幽灵（加 weight）
- **特征:** 暗晶花簇、水晶矿石、虚空草皮
- **结构:** 回声圣殿（Echo Sanctum）
- **特殊:** 暗晶花盛开事件在此触发
- **策略:** 寻找回声圣殿的关键群系

#### 🔥 虚空废土（Void Wastes）
- **温度/湿度:** 0.8 / -0.8
- **色调:** 红棕色调
- **地形:** 高地，高侵蚀
- **怪物:** 虚空蠕虫 + 暗晶幽灵
- **特征:** 水晶矿石、较少的植被
- **策略:** 高难度群系，适合刷矿

#### 🕳️ 暗晶洞穴（Crystal Caverns）
- **温度/湿度:** -0.7 / 0.7
- **色调:** 暗紫色调
- **地形:** 深层洞穴（depth=0.5）
- **怪物:** 虚空蠕虫 + 暗晶幽灵
- **特征:** 丰富的水晶矿石
- **特殊:** 虚空特有洞穴生成器（void_cave）
- **策略:** 矿工天堂，水晶矿石最密集

### 3.3 地形生成

- **噪声算法:** `minecraft:end/island` — 产生浮空岛屿地形
- **地表层:** void_grass_block（0-3 层） → void_dirt（3-6 层） → void_stone
- **洞穴:** 自定义 `void_echo:void_cave` + 原版洞穴/峡谷
- **矿石:** 水晶矿石（0-128 高度，虚空石替换）

```json

  "bossHealthMultiplier": 1.0,

}
```

| 参数 | 默认值 | 说明 |

## 💡 献给整合包作者的提示

1. **最早启动:** 虚空钥匙的石英配方无需下界合金，能让玩家更早进入虚空维度
2. **Boss 难度调节:** 通过 `config/void_echo.json` 的 `bossHealthMultiplier` 可自由调节
3. **虚空传送:** 在终虚渊中使用虚空传送门框架可安全返回主世界（记录传送位置）
4. **重生锚必带:** 终虚渊中床无效，必须带重生锚（可用虚空合金合成）
5. **暗晶精灵驯服:** 需要暗晶莓（从暗晶花随机掉落）→ 虚空触媒进化
6. **虚空锻炉互斥路径:** 回声线和裂隙线不能共存，需要玩家做出选择
7. **裂隙碎片:** 只有主动关闭裂隙才能获得，是被动的矿—需要靠近裂隙 3 秒
