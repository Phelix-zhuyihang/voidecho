Journey into the Void's End dimension, conquer the mighty Void Stalker, and uncover ancient secrets hidden within the Echo Sanctum.

## Overview

Void Echo: Ancient Sanctum is a Minecraft Fabric mod that adds a brand-new dimension — the **Void's End** — filled with unique biomes, structures, mobs, bosses, equipment, and a complete progression path.

## Features

- **New Dimension**: Void's End — a dark, mysterious dimension with alien landscapes
- **Two Biomes**: Void Plains (dark rolling hills) and Crystal Forest (glowing crystal spires)
- **Three Structures**: Forgotten Altar (Overworld), Void Fortress, Echo Sanctum (Void's End)
- **Five New Mobs**: Void Worm, Crystal Wraith, Shard Guard, Void Stalker (boss), Echo Warden (boss)
- **Two Boss Fights**: Multi-phase boss battles with unique mechanics
- **New Equipment**: Void Alloy armor set (4 pieces), Void Sword, Void Bow, Void Staff
- **Crafting System**: New materials, alloys, and recipes
- **Advancements**: 16 custom advancements guiding your progression
- **Custom Effects**: Void Touched (damage over time), Echo Resonance (damage amplifier)

## How to Start

1. **Find a Forgotten Altar** — These generate naturally in the Overworld. Look for stone brick structures with a portal frame.
2. **Mine Crystal Ore** — Found in the Overworld, drops Crystal Shards.
3. **Craft a Void Key** — Use Crystal Shards, an Ender Pearl, and Netherite Scrap.
4. **Activate the Portal** — Right-click the Void Portal Frame with the Void Key to enter the Void's End.

## Boss Guide

### Void Stalker
*Found in the Void Fortress.*

| Phase | Health | Behavior |
|-------|--------|----------|
| Phase 1 | 100%-66% | Hunts the player, teleports, melee attacks |
| Phase 2 | 66%-33% | Summons Void Worm minions, increased aggression |
| Phase 3 | 33%-0% | Frenzied rage, rapid teleportation, powerful combos |

**Drops**: Void Heart (1), Crystal Shards (3-8)

### Echo Warden
*Summoned by using a Void Heart on an Echo Altar surrounded by 4 Crystal Blocks.*

| Ability | Description |
|---------|-------------|
| Time Slow | Slows all nearby entities |
| Echo Slash | Devastating wide-area slash attack |
| Summon Shards | Calls Crystal Wraiths to assist |

**Drops**: Echo Core (1), Void Alloy Ingots (2-5), Crystal Shards (5-12)

## Equipment Guide

### Void Alloy Armor
- **Full Set Bonus**: Immunity to fall damage and void damage
- Each piece provides excellent protection comparable to Netherite

### Void Sword
- **Ability**: Right-click to teleport forward a short distance
- High damage output with void damage type

### Void Bow
- **Ability**: Arrows deal additional void damage
- High durability (500 uses)

### Void Staff
- **Ability**: Right-click to cast a Void Bolt projectile
- Void Bolt deals direct damage + AoE explosion + applies Void Touched effect

## Requirements

- Minecraft 1.21.1
- Fabric Loader >=0.15.0
- Fabric API >=0.102.0
- Java 21 or higher
- (Optional) ModMenu >=11.0.0

## Installation

1. Install Fabric Loader for Minecraft 1.21.1
2. Download Fabric API and place it in your `mods` folder
3. Download Void Echo: Ancient Sanctum and place it in your `mods` folder
4. Launch the game

# 虚空回响：远古圣殿

探索终虚渊维度，征服强大的虚空追猎者，揭开隐藏在回声神殿中的远古秘密。

## 概述

虚空回响：远古圣殿 是一个 Minecraft Fabric 模组，增加了一个全新的维度——**终虚渊**，包含独特的生物群系、结构、生物、Boss、装备和完整的进度路径。

## 特色

- **新维度**: 终虚渊 — 一个充满异域景观的黑暗神秘维度
- **两个生物群系**: 虚空平原（黑暗起伏的丘陵）和暗晶森林（发光的水晶尖塔）
- **三个结构**: 遗忘祭坛（主世界）、虚空堡垒、回声神殿（终虚渊）
- **五种新生物**: 虚空蠕虫、暗晶幽魂、碎片守卫、虚空追猎者（Boss）、回声守卫（Boss）
- **两场Boss战**: 多阶段Boss战斗，拥有独特机制
- **新装备**: 虚灵合金盔甲套装（4件）、虚灵剑、虚灵弓、虚空法杖
- **合成系统**: 新材料、合金和配方
- **进度系统**: 16个引导进度的自定义进度
- **自定义效果**: 虚空侵蚀（持续伤害）、回声共振（伤害放大）

## 如何开始

1. **寻找遗忘祭坛** — 在主世界自然生成，寻找带有传送门框架的石砖结构。
2. **挖掘暗晶矿石** — 在主世界生成，掉落暗晶碎片。
3. **合成虚空钥匙** — 使用暗晶碎片、末影珍珠和下界合金碎片。
4. **激活传送门** — 手持虚空钥匙右键点击虚空传送门框架，进入终虚渊。

## Boss攻略

### 虚空追猎者
*在虚空堡垒中找到。*

| 阶段 | 生命值 | 行为 |
|------|--------|------|
| 阶段1 | 100%-66% | 猎杀玩家，传送，近战攻击 |
| 阶段2 | 66%-33% | 召唤虚空蠕虫仆从，增加攻击性 |
| 阶段3 | 33%-0% | 狂怒状态，快速传送，强力连击 |

**掉落**: 虚空之心(1)、暗晶碎片(3-8)

### 回声守卫
*在回声祭坛四周放置4个暗晶块，使用虚空之心右键祭坛召唤。*

| 能力 | 描述 |
|------|------|
| 时间减缓 | 减缓附近所有实体速度 |
| 回声斩击 | 毁灭性的范围斩击攻击 |
| 召唤碎片 | 召唤暗晶幽魂协助战斗 |

**掉落**: 回声核心(1)、虚灵合金锭(2-5)、暗晶碎片(5-12)
