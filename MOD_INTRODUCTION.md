# Void Echo: Ancient Sanctum — 虚空回响：远古圣殿

**模组版本 1.0.0 · Minecraft 1.21.1 · Fabric · 中文/English**

---

## 一、概述

Void Echo 是一个以"虚空"为主题的维度探索与 Boss 挑战模组。玩家在主世界发现被遗忘的远古祭坛，打造虚空钥匙，激活传送门进入 **终虚渊 (Void's End)** —— 一个永恒午夜、紫雾弥漫的异次元。在那里，你将面对被虚空腐化的守护者、收集失落的水晶碎片、锻造虚空合金装备，并最终揭开艾洛斯 (Aerolith) 远古文明的真相。

### 核心玩法循环

```
主世界 → 发现遗忘祭坛 → 收集暗晶碎片 → 制造虚空钥匙 → 激活传送门
  ↓
进入终虚渊 → 探索 4 个群系 → 寻找虚空要塞 → 击败虚空追猎者 (Boss #1)
  ↓
获得虚空之心 → 制造虚空合金装备 → 发现回声圣殿 → 召唤回声守卫 (Boss #2)
  ↓
获得回声核心 → 虚空锻炉升级 → 关闭虚空裂隙 → 终局挑战
```

## 二、快速上手

### 第一步：收集暗晶碎片
在主世界地下开采 **暗晶矿石** (Crystal Ore)，获得 **暗晶碎片** (Crystal Shard)。这是一种半透明紫晶矿物，生成于 Y=-16 到 Y=64 之间。

### 第二步：寻找遗忘祭坛
在主世界的平原和沙漠群系中，寻找 **遗忘祭坛** (Forgotten Altar) —— 一个 7×7 圆形石砖平台，四角水晶柱，中心虚空传送门框架。

### 第三步：制造虚空钥匙
使用暗晶碎片合成 **虚空钥匙** (Void Key)。手持钥匙右键点击祭坛中心的传送门框架即可激活传送。

### 第四步：进入终虚渊
传送门激活后，你将进入终虚渊——永夜维度。床不能使用，但**重生锚**可以。探索四个群系，收集 5 块回声记忆碎片拼凑真相。

## 四、生物群系（4 个）

### 虚空平原 (Void Plains)
广阔平坦的紫色荒原。怪物密集，虚空追猎者在此出没。生成：Void Worm (100权重)、Crystal Wraith (50权重)、Void Stalker (5权重)。

### 水晶森林 (Crystal Forest)
水晶簇遍布的紫色林地，水晶花在暗夜中发出微光。回声圣殿隐藏于此。生成：Crystal Wraith (80权重)、Void Worm (40权重)、Crystal Sprite (5权重)。

### 虚空荒原 (Void Wastes)
暗红紫色废土，虚空能量最浓的区域。虚空裂隙生成于此。生成：Void Worm (120权重)、Crystal Wraith (60权重)。特有结构：Void Rift Core。

### 水晶洞窟 (Crystal Caverns)
地下水晶洞窟，大量水晶矿脉暴露于洞壁。艾洛斯人最后的避难所遗迹。生成：Crystal Wraith (80权重)、Shard Guard (50权重)。

## 五、结构（4 个）

### 遗忘祭坛 (Forgotten Altar) — 主世界
- 生成群系：平原、沙漠、向日葵平原，间距 40 区块
- 7×7 圆形平台 + 4 水晶柱 + 中心传送门框架
- 25% 概率平台下藏有战利品箱
- 包含：回声记忆碎片 I + 讲台成书《最后一位建筑师的手记》

### 虚空要塞 (Void Fortress) — 终虚渊
- 生成群系：虚空平原、水晶森林，间距 30 区块
- 入口塔楼 + 螺旋楼梯 + 主走廊 + Boss 房间
- 兵营 + 金库（双箱子）+ 陷阱房
- 包含：回声记忆碎片 III/IV/V + 讲台成书《禁卫队长的最后记录》

### 回声圣殿 (Echo Sanctum) — 终虚渊
- 生成群系：水晶森林，间距 50 区块
- 9×9 花纹平台 + 4 根漂浮水晶柱 + 拱门
- 中心回声祭坛，4 个水晶块已就位可直接召唤
- 包含：回声记忆碎片 II + 讲台成书《召唤仪典》+ 2 个隐藏战利品箱

### 虚空裂隙核心 (Void Rift Core) — 终虚渊
- 生成群系：虚空荒原，间距 25 区块
- 地下圆形密室（半径 5）+ 中心活跃裂隙方块
- 4 个水晶掩体柱 + 入口竖井
- 裂隙方块周期性脉冲粒子，潜行右键关闭
- 关闭掉落：Rift Core ×1 + Rift Fragment ×2-3
- 10% 概率触发 Void Shade 迷你 Boss

## 十二、世界观背景

远古文明 **艾洛斯 (Aerolith)** 曾在此繁荣。他们发现水晶可以储存记忆——这是他们最伟大的发明，也是最终的灭亡。因为**虚空以记忆为食**。

国王 **艾里昂四世 (Aerion IV)** 将自身意识封入巨型水晶以压制虚空入口——他就是 **回声守卫 (Echo Warden)**。他的禁卫队长 **凯伦 (Kaelen)** 为保护国王主动吸收虚空能量，被腐化为 **虚空追猎者 (Void Stalker)**。

5 块回声记忆碎片记录了这个故事。击败回声守卫意味着封印完全解除——虚空不再被压制。结局文本 "代价是什么？" 指向一个现实：你终结了一个循环，但开启了另一个。

*完整世界观设定见 `docs/worldbuilding/voids_end.md`。*
